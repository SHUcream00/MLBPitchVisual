# MLBPitchVisual

[![Build Status](https://travis-ci.org/SHUcream00/MLBPitchVisual.svg?branch=master)](https://travis-ci.org/SHUcream00/MLBPitchVisual)

Raw data by [Baseballsavant](https://baseballsavant.mlb.com/)
